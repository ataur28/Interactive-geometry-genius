Question And Answer

1. What is DOM and what is the purpose of DOM?

Answer: The Document Object Model (DOM) is a programming interface for web documents.

It defines the logical structure of documents and the way to access and manipulate a document.

2. How will you select HTML elements using DOM? Name the DOM methods?

Answer: we can use various methods depending on the specific requirements of our code.so some common way to select element

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Answer: Event bubbles are a concept in the Document Object Model. This occurs when an element receives an event, and bubbles up its parent and ancestor elements in the DOM tree until that event reaches the parent element.

4. What is a callback function and why will you use it?

Answer: A callback function is a function that is passed as an argument to another function, which is called to perform some sort of routine or action inside the outer function.
